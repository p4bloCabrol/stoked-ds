# Design Tokens

stoked-ds uses **CSS custom properties** (variables) as design tokens. All tokens are prefixed with `--stoked-` and can be imported for use in your own styles.

## Importing Tokens

```css
@import 'stoked-ds/dist/tokens/index.css';
```

This imports all token categories: colors, spacing, typography, borders, shadows, transitions, and z-index.

### Easing Functions

| Token | Curve |
|-------|-------|
| `--stoked-easing-default` | `cubic-bezier(0.4, 0, 0.2, 1)` |
| `--stoked-easing-in` | `cubic-bezier(0.4, 0, 1, 1)` |

| `--stoked-easing-bounce` | `cubic-bezier(0.68, -0.55, 0.265, 1.55)` |

### Presets

| Token | Animates |
|-------|----------|
| `--stoked-transition-colors` | color, background, border, shadow |
| `--stoked-transition-transform` | transform |
| `--stoked-transition-all` | all properties |

```css
.my-button:hover {
  transition: var(--stoked-transition-colors);
  background: var(--stoked-color-primary-hover);
}
```
